# NS DB Viewer

## 프로그램 소개

**NS DB Viewer**는 REMOTE2.NSDB 데이터베이스의 테이블을 조회할 수 있는 Windows Forms 기반 데이터베이스 뷰어 프로그램입니다.

이 프로그램은 **사용자 모드**와 **관리자 모드** 두 가지 모드로 운영되며, 관리자가 사용자에게 공유할 테이블을 선택적으로 관리할 수 있습니다.

---

## 주요 기능

### 📊 데이터베이스 조회
- REMOTE2.NSDB 데이터베이스에 연결
- 테이블 목록 확인
- 선택한 테이블의 전체 데이터 조회
- 실시간 데이터 그리드 표시

### 👤 사용자 모드
- 관리자가 허용한 테이블만 조회 가능
- 읽기 전용 액세스
- 안전한 데이터 열람 환경

### 🔑 관리자 모드
- 전체 테이블 목록 확인
- 사용자에게 공유할 테이블 선택 및 관리
- 테이블 추가/제거 권한

---

## 사용 방법

### 1️⃣ 프로그램 실행
프로그램을 실행하면 기본적으로 **사용자 모드**로 시작됩니다.

### 2️⃣ 사용자 모드에서 테이블 조회
1. 왼쪽 목록에서 조회하고 싶은 테이블을 선택합니다.
2. 오른쪽 그리드에 해당 테이블의 데이터가 표시됩니다.

### 3️⃣ 관리자 모드 진입

#### 관리자 비밀번호
```
admin123
```

#### 로그인 절차
1. 우측 상단의 **🔐 Admin Login** 버튼을 클릭합니다.
2. 비밀번호 입력 창에 `admin123`을 입력합니다.
3. 확인을 클릭하면 관리자 모드로 전환됩니다.

### 4️⃣ 관리자 모드에서 테이블 관리

관리자 모드에서는 다음 작업을 수행할 수 있습니다:

#### ➕ 사용자에게 테이블 추가
1. 왼쪽 목록에서 사용자에게 공유할 테이블을 선택합니다.
2. 하단의 **➕ Add for Users** 버튼을 클릭합니다.
3. 해당 테이블이 사용자 모드에서 조회 가능하게 됩니다.

#### ➖ 사용자에게서 테이블 제거
1. 왼쪽 목록에서 제거할 테이블을 선택합니다.
2. 하단의 **➖ Remove** 버튼을 클릭합니다.
3. 해당 테이블이 사용자 모드에서 숨겨집니다.

### 5️⃣ 사용자 모드로 전환
관리자 모드에서 우측 상단의 **↩️ Switch to User** 버튼을 클릭하여 언제든지 사용자 모드로 전환할 수 있습니다.

## 화면 구성

### 상단 영역
- **왼쪽**: 연결 상태 및 현재 작업 정보 표시
- **오른쪽**: 현재 모드 표시 (👤 사용자 모드 / 🔑 관리자 모드)
- **우측 상단**: 모드 전환 버튼

### 중앙 영역
- **왼쪽 패널**: 테이블 목록
- **오른쪽 패널**: 선택한 테이블의 데이터 그리드

### 하단 영역 (관리자 모드 전용)
- **➕ Add for Users**: 선택한 테이블을 사용자에게 공유
- **➖ Remove**: 선택한 테이블을 사용자 목록에서 제거

## 기술 정보

### 데이터베이스 연결 정보
- **서버**: ASUS\MONITORTF
- **데이터베이스**: REMOTE2.NSDB
- **인증**: SQL Server 인증

### 개발 환경
- **.NET**: 8.0
- **UI 프레임워크**: Windows Forms
- **데이터베이스 라이브러리**: Microsoft.Data.SqlClient

### 구성 파일
사용자 설정은 `user_config.json` 파일에 자동으로 저장됩니다.
- **위치**: 프로그램 실행 폴더
- **내용**: 사용자에게 허용된 테이블 목록

## 문제 해결

### 데이터베이스 연결 실패
- 데이터베이스 서버가 실행 중인지 확인
- 네트워크 연결 상태 확인
- SQL Server 인증 정보 확인

### 테이블이 표시되지 않음 (사용자 모드)
- 관리자 모드로 전환하여 테이블이 사용자에게 공유되었는지 확인
- 관리자에게 테이블 액세스 권한 요청

### 관리자 로그인 실패
- 비밀번호가 정확한지 확인: `admin123`
- 대소문자 구분 확인
